# Invepinca - Web React + Tailwind

Sitio web profesional de impermeabilización con calculadora de presupuesto y formulario de WhatsApp integrado.

## 🚀 Instalación
```bash
npm install
npm run dev
```

## 🧩 Tecnologías
- React 18 + Vite
- Tailwind CSS
- Framer Motion (animaciones)

## 🛠 Configuración
Edita el archivo `src/InvepincaApp.jsx` para ajustar:
- Teléfono de WhatsApp (constante `PHONE`)
- Precios y rendimientos

## 🌐 Despliegue
Compatible con Vercel, Netlify o GitHub Pages.
